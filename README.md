# Bike Escape
Mobile app for city games and bike races. 

## API
Backend services.
- Java
- Spring-Boot
- REST
- JPA
- PostgreSQL
- Docker
- Lombok
- Swagger
- JWT

## MOBILE
Hybrid mobile app for end users.
- Ionic 3
  
## WEB
Web admin panel for managing content.
- Angular
- TypeScript
- Material UI

## STATIC
Static web page for app promotion purposes.
